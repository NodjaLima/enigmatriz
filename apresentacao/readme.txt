Documentação ENIGMATIX

Escolhido em grupo o Tema "Aventura no Egito", envolvendo charadas em uma piramide, com três personagens em jornada individual.

Iniciei o projeto desenhando o diagrama do processo do jogo até chegar á vitória ou gameover. Determinei questões importantes do ponto de decisão, como a quantidade de opções que o jogador teria para responder, se teriam chances para errar ou não. A partir desse desenho, consegui entender que funções seriam necessárias em Javascript para que a jogabilidade acontecesse.


Como um unico erro já leva ao Gameover, optamos por usar a estrutura de repetição WHILE e dentro dela foi implementado a condição IF. 

Para, ao final de cada fase, chamar a fase seguinte, dentro das funções foi colocado window.location.

Para que o promtp aparecesse a partir de uma interação do usuário, foi utilizado onclick em imagens.

As perguntas foram frutos de pesquisas na web. 

Depois de feita a função da primeira fase do primeiro personagem, replicar o codigo foi simples.

Fotos de background e textos, foram decididos a partir das preferencias individuas do desenvolvedor, a partir do tema.

Durante todo o desenvolvimento do jogo, contei com o auxílio tecnico e psicologico dos meus companheiros de equipe, denominada 'Grupo 8'. Em nenhum momento houve desrespeito no grupo, apesar de algumas discordancias que renderam conversas que contribuíram muito para a realização de todos os jogos.
